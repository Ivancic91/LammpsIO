Installation Instructions
=========================

1. Clone or download the LammpsIO project at https://github.com/Ivancic91/LammpsIO

2. Change directory into the LammpsIO project and copy src/LammpsIO.py to a directory in your $PYTHONPATH

3. If you have not already, install the netCDF4 library for python. You may do this with the package manager Anaconda: 'conda install -c esri netcdf4'. Note if you install another version of netCDF4 you may mess up installation of BASECODE.
